## Introdução
Este programa simula operações básicas de uma conta bancária. Ideal para fins didáticos, explorando conceitos de orientação a objetos.

## Tecnologias usadas

[![Tecnologias](https://skillicons.dev/icons?i=git,github,cs,dotnet,visualstudio)](https://skillicons.dev)

## Funcionalidades

- **Criação de Conta Corrente**
- **Depósito em Conta**
- **Saque com Validação de Saldo**
- **Exibição de Saldo Atual**
- **Histórico de Operações**


## Estrutura do Projeto
- `CriarID` - Contém o objeto `CriarID` onde é gerado um ID aleatório para a conta que será criada.
- `ContaCorrente` – Contém o objeto `ContaCorrente` com os principais métodos e atributos da conta.
- `Program.cs` – Ponto de entrada principal da aplicação, responsável pelo fluxo do programa.


## Como Usar

### Requisitos

- .NET SDK (.NET 8.0 ou superior recomendado)

### 1. Clone o repositório.
 
```
git clone https://github.com/AgathaSates/Conta-Corrente.git
```
### 2. Abra o terminal ou o prompt de comando e navegue até a pasta raiz

```
cd cd Conta-Corrente
```

### 3. Utilize o comando abaixo para restaurar as dependências do projeto.

```
dotnet restore
```

### 4. Navegue até a pasta do projeto

```
cd cd Conta-Corrente.ConsoleApp
```

### 5. Execute o projeto

```
dotnet run
```
